Chapter 2: Polynomials

Zeros, Division Algorithm, Factor Theorem, and Factorization

Definitions

1. Polynomial

Polynomial: An algebraic expression of the form:

p(x) = a₀ + a₁x + a₂x² + ... + aₙxⁿ

where a₀, a₁, ..., aₙ are constants and n is non-negative integer

2. Degree of Polynomial

Degree: The highest power of the variable in the polynomial.

Example: In p(x) = 3x³ + 2x² - 5x + 7, degree = 3

3. Types of Polynomials by Degree

  • Constant Polynomial: Degree 0. Example: p(x) = 5
  • Linear Polynomial: Degree 1. Example: p(x) = 2x + 3
  • Quadratic Polynomial: Degree 2. Example: p(x) = x² - 4x + 3
  • Cubic Polynomial: Degree 3. Example: p(x) = x³ + 2x² + x + 1

4. Zero of a Polynomial

Zero: Value of x for which p(x) = 0

Example: For p(x) = x² - 4, zeros are x = 2 and x = -2 (since 2² - 4 = 0)

Important Theorems

1. Remainder Theorem

Statement: If p(x) is a polynomial and (x - a) divides it, then the remainder when p(x) is divided by (x - a) is p(a).

Mathematical form: p(x) = q(x)(x - a) + p(a)

Example: p(x) = x² - 3x + 2, find remainder when divided by (x - 1)

Remainder = p(1) = 1 - 3 + 2 = 0

2. Factor Theorem

Statement: (x - a) is a factor of p(x) if and only if p(a) = 0

Proof: If (x - a) is factor, then p(x) = (x - a) × q(x)

Putting x = a: p(a) = 0

Example: Is (x - 2) a factor of p(x) = x³ - 2x² + x - 2?

p(2) = 8 - 8 + 2 - 2 = 0 ✓ Yes, it is a factor

3. Algebraic Identity for (a + b)² and (a - b)²

(a + b)² = a² + 2ab + b²

(a - b)² = a² - 2ab + b²

(a + b)(a - b) = a² - b²

Important Formulas

1. For quadratic ax² + bx + c:

Sum of zeros = -b/a

Product of zeros = c/a

2. For cubic ax³ + bx² + cx + d:

Sum of zeros = -b/a

Sum of products of pairs = c/a

Product of zeros = -d/a

3. Factorization: If roots are α and β:

ax² + bx + c = a(x - α)(x - β)

4. (a + b)³ = a³ + 3a²b + 3ab² + b³

(a - b)³ = a³ - 3a²b + 3ab² - b³

a³ + b³ = (a + b)(a² - ab + b²)

a³ - b³ = (a - b)(a² + ab + b²)

Methods & Examples

Finding Zeros of Quadratic

Example: Find zeros of p(x) = x² - 5x + 6

Method 1 (Factorization): x² - 5x + 6 = (x - 2)(x - 3) = 0

∴ x = 2 or x = 3

Method 2 (Quadratic formula):

x = (5 ± √(25-24))/2 = (5 ± 1)/2

x = 3 or x = 2

Using Remainder Theorem

Example: Find remainder when x³ - 3x² + 2x - 5 is divided by (x - 2)

Let p(x) = x³ - 3x² + 2x - 5

By Remainder Theorem: Remainder = p(2)

p(2) = 8 - 12 + 4 - 5 = -5

∴ Remainder = -5

Using Factor Theorem

Example: Check if (x - 1) is a factor of p(x) = x³ - 6x² + 11x - 6

By Factor Theorem, (x - 1) is factor if p(1) = 0

p(1) = 1 - 6 + 11 - 6 = 0 ✓

Yes, (x - 1) is a factor

Relation Between Zeros and Coefficients

Example: If zeros of 2x² - 5x + 3 are α and β, find α + β and αβ

For ax² + bx + c:

α + β = -b/a = -(-5)/2 = 5/2

αβ = c/a = 3/2
